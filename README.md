# 🔢 React Counter App

A simple, elegant, and interactive Counter Application built using **React** and **Vite**. This project demonstrates fundamental React concepts like state management (`useState`) and event handling in a clean, modern UI.

## 🚀 Features

- **Increment**: Increase the counter value by 1.
- **Decrement**: Decrease the counter value by 1.
- **Reset**: Instantly reset the counter back to 0.
- **Modern UI**: Clean and responsive styling with custom CSS.
- **Lightning Fast**: Powered by Vite for instant server start and lightning-fast HMR (Hot Module Replacement).

## 🛠️ Tech Stack

- **Frontend Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS

## 📋 Prerequisites

Before you begin, ensure you have installed:
- [Node.js](https://nodejs.org/) (v16.0 or higher recommended)
- npm (usually comes with Node.js)

## 💻 Running Locally

Follow these steps to run the project on your local machine:

1. **Navigate to the project directory** (if you aren't already there):
   ```bash
   cd counter-app
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and visit the URL shown in your terminal (usually `http://localhost:5173/`).

## 📁 Project Structure

- `src/counter/counterapp.jsx`: The main component containing the counter logic and UI.
- `src/counter/CounterApp.css`: The stylesheet for the counter component.
- `src/App.jsx`: The root component that renders the application.
- `src/main.jsx`: The entry point of the React application.
